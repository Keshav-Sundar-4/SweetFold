## Inference instructions
1. Download the SweetFold epoch 23 checkpoint from https://huggingface.co/Keshav-Sundar-4/SweetFold/tree/main
2. Download Boltz v1.0.0 using the command: pip install "boltz[cuda]==1.0.0"
3. Traverse to the 'boltz' folder containing Boltz source code. Replace this folder with SweetFold/src/boltz
4. Run SweetFold. An example shell script for HPC usage is in this folder as well
